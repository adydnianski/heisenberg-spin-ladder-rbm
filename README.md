This code allows you to use restricted boltzmann machine approach to calculate the ground state of a system of heisenberg spins on a ladder.
We are basing on an article by Carleyo and Troyer published in Science in 2017: Carleo, G., & Troyer, M. (2017). Solving the quantum many-body problem with artificial neural networks. Science, 355(6325), 602-606.
